# 2048 Pro

A premium version of the classic 2048 game with multiple game modes, themes,
analytics, and leaderboards.

## Tech Stack
- Frontend: React 18, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT (Guest-first)

## Features (Planned)
- Multiple game modes
- Theme system
- Guest & authenticated users
- Analytics dashboard
- Global leaderboards
